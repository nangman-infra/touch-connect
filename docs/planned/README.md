# Planned Docs

`docs/planned/`는 아직 계획 중이거나 초안이며, current source of truth로 승격되지 않은 문서를 두는 곳이다.  
실무적으로는 `미참조 상태`의 작업 문서를 두는 디렉터리다.

## 무엇을 두는가

- 초기 아이디어 정리
- 아직 검토되지 않은 구조 초안
- 회의용 메모
- 이후 active로 승격될 수 있는 설계 문서

## 무엇을 두지 않는가

- 현재 계약 문서
- 현재 기준 리서치 문서
- 구현이 바로 참조해야 하는 문서

## 규칙

- 이 README는 `planned 문서`가 아니라 `planned 디렉터리의 구조 인덱스`이므로 예외적으로 active registry에 등록될 수 있다.
- 아래 규칙은 `docs/planned/` 아래에 실제로 놓이는 일반 planned 문서에 적용된다.
- planned 문서는 active 인덱스에 올리지 않는다.
- planned 문서는 현재 정책처럼 단정적으로 쓰지 않는다.
- active 문서에서 planned 문서를 언급할 때는 반드시 `planned` 또는 `초안`이라고 표시한다.
- active로 승격되면 `docs/active/`로 이동하고 `docs/active/README.md`에 등록한다.
- 폐기되면 `docs/archive/`로 이동한다.

## 현재 상태

현재 planned 문서는 없다.
