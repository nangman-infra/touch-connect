# Archive Docs

`docs/archive/`는 더 이상 현재 기능 구현이나 운영 판단의 기준으로 직접 참조되지 않는 문서를 보관하는 곳이다.

## archive 대상

- 완료된 단계 문서
- superseded 된 설계 문서
- 더 이상 직접 참조하지 않는 리서치 문서
- 폐기되었지만 기록이 필요한 초안

## archive 규칙

- 이 README는 `archive 문서`가 아니라 `archive 디렉터리의 구조 인덱스`이므로 예외적으로 active registry에 등록될 수 있다.
- archive 문서는 현재 기준 문서가 아니다.
- archive 문서 상단에는 archive note를 남긴다.
- archive note에는 최소한 아래를 적는다.
- archive 문서는 historical reference로만 쓴다.
- archive 문서를 다시 current contract로 되돌릴 때는 planned 또는 active 문서로 재발행한다.

```md
> Archived: YYYY-MM-DD
> Reason: 왜 archived 되었는지
> Superseded by: 현재 기준 문서 경로 또는 `none`
```

- archive로 이동할 때 active 인덱스와 교차참조를 같이 갱신한다.

## 현재 상태

현재 archive 문서는 아직 없다.
